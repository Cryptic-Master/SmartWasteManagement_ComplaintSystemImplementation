The Smart Waste
Collection System is a digital platform designed to help municipalities manage
garbage collection efficiently and maintain city cleanliness. The system
monitors smart bins distributed throughout the city, each equipped with a
sensor that measures the fill level. When a bin reaches a predefined threshold,
the sensor automatically sends a notification to the central system. Based on
these notifications, the system generates optimal daily routes for each driver,
assigning a specific set of bins that they are responsible for emptying.
Drivers must log in to the system to access their daily tasks, and after
emptying a bin, they confirm the action within the application. The system
verifies the bin's status through the sensor reading; if the bin is indeed
empty, the task is recorded as completed. If the bin is still full despite the
driver’s confirmation, the system automatically generates a complaint, marking
the bin as pending until the issue is resolved. A technician is assigned to
inspect the bin and resolve the problem. If the sensor or the bin is found to
be faulty, no penalty is applied to the driver; however, if the bin is
functioning correctly and the driver did not empty it, the system records a
penalty. Citizens can also submit complaints if a bin is broken, full, or has a
cleanliness issue. Supervisors can view all complaints, whether
pending, resolved, or rejected, and assign them to technicians for handling.
Technicians access the system to view their assigned complaints, resolve
issues, and submit resolution reports. All users must log in to the system to
perform their actions.
